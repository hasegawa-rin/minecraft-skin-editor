# SPEC - 技術仕様・要件定義

## 機能要件（v1）
1. **2D編集キャンバス**: 面ごとの分割表示。各パーツの6面（まえ/うしろ/ひだり/みぎ/うえ/した）をラベル付きで表示
2. **描画ツール**: ブラシ、バケツ（面制約付き）、スポイト、消しゴム
3. **カラーパレット**: カラーピッカー、HEX入力、コピー機能、透明モード（オン/オフ）
4. **レイヤー**: base（下地）とoverlay（重ね着）の2レイヤー切り替え
5. **パーツ選択**: ぜんたい表示（体の形配置）と個別パーツ表示の切り替え
6. **3Dプレビュー**: スキンデータのリアルタイムプレビュー（レイヤー表示モード切替対応）
7. **ファイル操作**: PNGインポート（64x64 / 64x32対応）、PNGエクスポート（64x64フォーマット）
8. **操作履歴**: undo/redo（最大20件）
9. **ズーム**: ボタン式（−/倍率表示/＋）、倍率クリックで100%リセット
10. **モデル対応**: Classic（腕幅4px）とSlim（腕幅3px）

## 非機能要件
- **ターゲットユーザー**: 小学校低学年（1-2年生）
- **言語**: 全UIテキストは日本語。小3以上の漢字にはふりがな。英語はカタカナ表記
- **デザイン**: フラットデザイン、ナチュラルライトテーマ、12pxグリッド基準
- **動作環境**: モダンブラウザ（Chrome推奨）
- **ローカル動作**: サーバー不要、全てブラウザ内で完結

## 技術構成
- React 19 + TypeScript
- Vite 8（開発サーバー＋ビルド）
- Three.js + React Three Fiber + Drei（3Dプレビュー）
- Zustand（状態管理）
- スキンデータ：64x64px RGBA（Uint8ClampedArray、16384要素）
