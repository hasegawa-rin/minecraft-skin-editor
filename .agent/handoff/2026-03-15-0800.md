## プロジェクトの現在の状態
- **動作する状態**: `npm run dev` でローカル開発サーバーが起動し、ブラウザで操作可能
- **2D編集**: 面ごとに分割された直感的なキャンバス。各面にラベル（まえ/うしろ/ひだり/みぎ/うえ/した）付き
- **ツール**: ブラシ、バケツ（面制約付き）、スポイト、消しゴム
- **透明モード**: オン/オフの2値。マイクラBedrock Editionの仕様に合わせた実装
- **3Dプレビュー**: placeholder状態（箱だけ表示）。本格実装は未着手
- **PNG出力**: 64x64フォーマットで正常にエクスポート可能

## 試したこと・結果
- CSS `paddingLeft` での展開図揃え → FaceCanvasのラベル幅が不定で位置ずれ → CSS Gridに変更して解決
- `position: sticky` でズームボタン配置 → スクロールコンテナ内では中央に固定されてしまう → ZoomControlsを分離してeditor-mainに直接配置（position: absolute）で解決
- `useState` でhex入力管理 → スポイトでの色変更が同期しない → `useRef + useEffect` に変更して解決
- 透明度スライダー（0-255） → マイクラBEでは半透明不可のため無意味 → オン/オフトグルに簡素化
